Kai Chronicles

v1.3
This is a game books player for Lone Wolf books 1 to 4.

Only latest versions of Firefox and Chrome are supported. Other browsers and versions are not supported (they may work... or not).

Keep in mind that if you clean your browser history, the current game state will be lost. You can save the current game to a file at Settings > Save game.

You can play this as an Android app (mininum version 4.4.2). Download it here (14 MB).

Have fun!
