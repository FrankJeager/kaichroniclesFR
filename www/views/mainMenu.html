Kai Chronicles

v1.1
This is a game player for Lone Wolf game books. Books 1 & 2 are implemented.
Keep in mind that if you clean your browser history, the current game state will be lost. You can save the current game at Settings > Save game.
You can play this as an Android app (min. Android version supported is 4.4.2). Download it here (about 10MB).
 Have fun!
