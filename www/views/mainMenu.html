Kai Chronicles

v1.1
This is a game player for Lone Wolf game books. Only books 1 & 2 are implemented.

Only latest versions of Firefox and Chrome are supported. Other browsers and versions are not supported (maybe they work... or not).

Keep in mind that if you clean your browser history, the current game state will be lost. You can save the current game at Settings > Save game.

You can play this as an Android app (min. Android version supported is 4.4.2). Download it here (about 10 MB).

Have fun!
